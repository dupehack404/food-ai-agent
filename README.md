# Food AI Agent

Multi-agent pet project for automated food planning and ordering.

## Features
- user preference extraction
- meal planning
- delivery window decision
- order planning
- Telegram bot interface
- JSON persistence

## Agents
- PreferenceAgent
- MealAgent
- DeliveryWindowAgent
- OrderAgent

## Stack
- Python
- aiogram
- OpenAI API
- Playwright
- JSON storage

## Run
```bash
python -m app.bot
